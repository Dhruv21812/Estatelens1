# EstateLens AI — The Brutally Honest Brand Auditor

An elite AI-powered branding audit tool for real estate developers. Built to identify blind spots, benchmark against competitors, and provide actionable growth strategies.

## Features

- **Competitor Comparison**: Benchmark your brand against any competitor to identify market gaps.
- **Trust Analysis**: Deep dive into signals that build or break developer credibility.
- **Conversion Audit**: Pinpoint exactly where you are losing potential homebuyers.
- **Marketing Edge**: Viral-ready reel ideas and high-converting ad hooks.
- **Brutal Truth**: A "no-BS" strategy breakdown from an AI luxury branding consultant.
- **Export & Share**: Professional PDF reports and shareable audit links.
- **Demo Mode**: Full functional preview without needing an API key.

## Tech Stack

- **Framework**: [Next.js](https://nextjs.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)

## Getting Started

1. Clone the repository.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
4. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Deployment

This version is optimized for a simple UI demo deployment on Vercel. No API keys are required as it uses mock analysis data to demonstrate the premium branding audit experience.
